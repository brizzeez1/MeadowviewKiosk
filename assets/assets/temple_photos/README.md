# Temple Photos for Screensaver

This folder contains temple photos for the kiosk screensaver.

## How to Add Photos

1. Add your own temple photos to this folder
2. Use JPG, JPEG, PNG, or WebP format
3. Recommended size: 1920x1080 pixels (Full HD)
4. Update `config/config.js` to include your photo filenames:

```javascript
TEMPLE_PHOTOS: [
    "assets/temple_photos/salt_lake_temple.jpg",
    "assets/temple_photos/provo_temple.jpg",
    "assets/temple_photos/your_temple.jpg"
]
```

## Photo Guidelines

- Use high-quality, high-resolution images
- Landscape orientation works best
- Beautiful lighting (golden hour, twilight) looks great
- Make sure you have rights to use the images

## Where to Find Temple Photos

- Church media library: https://www.churchofjesuschrist.org/media
- Official temple photos from the Church are available for non-commercial use
